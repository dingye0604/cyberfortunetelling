# 周易命理查询

纯静态八字排盘网页，基于周易和中国传统文化。输入公历生日，输出四柱八字、五行分析、纳音、生肖、梅花易数卦象。

## 功能

- **八字排盘**：年柱（立春分界）、月柱（五虎遁）、日柱（查表推算）、时柱（五鼠遁）
- **五行分析**：八字五行分布统计、日主旺衰判断
- **纳音五行**：六十甲子纳音查询
- **生肖**：年柱地支对应生肖
- **梅花易数**：数字起卦，含本卦变卦

## 技术栈

纯静态 HTML + CSS + JavaScript，无依赖、无构建步骤、无后端。

## 本地预览

```bash
npx serve .
```

## 部署

推送到 GitHub，Pages 设置为 `main` 分支 `/` 目录。
