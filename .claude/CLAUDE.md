# 周易命理查询工具

基于周易和中国传统文化的生辰八字查询静态网页。纯前端，无后端无数据库。

## 约束

- **纯静态**：没有服务器、数据库、后端 API、构建步骤
- **数据全员硬编码**：所有查询表（天干地支、纳音六十甲子、节气、六十四卦）直接写在 JS 中，不额外加载 JSON
- **GitHub Pages 部署**：推送到 GitHub 即可发布

## 目录结构

```
/
├── index.html          # 首页（输入表单）
├── css/
│   └── style.css       # 样式（尽量不依赖框架）
├── js/
│   ├── bazi.js         # 八字排盘：公历→四柱（年柱立春分界，月柱节气分界，日柱查表）
│   ├── wuxing.js       # 五行分析：天干地支五行属性、生克、旺衰
│   ├── gua.js          # 起卦：基于数字的简易梅花易数
│   ├── data.js         # 所有静态数据：天干地支、纳音表、节气表、日柱基准表、六十四卦
│   └── app.js          # UI 绑定：表单处理、渲染结果
├── assets/
│   └── (icons 等静态资源，如有)
└── README.md
```

## 工程纪律

- **改完必验**：用 `npx serve` 本地预览确认功能正常
- **数据先行**：所有预测/推算规则必须配原始参考数据，不写空对空的逻辑
- **日柱查表精度**：日柱用基准表（每年 1 月 1 日的日干支）+ 60 天周期推算，覆盖 1900-2100 年，节气边界用固定日期近似表（偏差 ±1 天可接受）
- **提交前格式化**：HTML/CSS/JS 保持整洁，无调试日志

## 本地预览

```bash
npx serve .
```

## 部署

推送到 GitHub 即可，GitHub Pages 设置为 `main` 分支 `/` 目录。
